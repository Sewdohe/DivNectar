# CraftNectar

I finally made a Minecraft server! 

I've been wating to do this for years and years! It feels so good to have finally done it.

## Connection

The server is compatible with the following clients:
  * Minecraft Vanilla 1.19.2
  * Minecraft Fabric Client 1.19.2


### **The server url is:** 

`craft.divnectar.com`

## Features/Plugins

### Realtime Map
The server is equipped with a real-time map generator. You can access it at [the servers url](https://craft.divnectar.com:8123 "click to view DynMap")

View the [DynMap Wiki](https://github.com/webbukkit/dynmap/wiki "DynMap Wiki")

### PVP Toggle
You can toggle PVP on/off with `/pvp`

### Factions Mod
View the [Wiki](https://github.com/ickerio/factions/wiki "Faction's Wiki")
